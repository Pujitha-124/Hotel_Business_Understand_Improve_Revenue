# Milestone 1 – Data Modeling & Ingestion (Power BI)

## Objective
To build a star schema data model for a hotel business dataset and perform required data transformations using Power BI.

## Dataset
- Source: Provided hotel_data.csv
- Tool Used: Power BI Desktop

## Tasks Completed
- Loaded CSV dataset into Power BI
- Performed data cleaning and transformations in Power Query
- Created calculated columns:
  - Booking Duration
  - Room Category
  - Stay Type (Short / Medium / Long)
- Built Star Schema with:
  - Fact Table: Fact_Bookings
  - Dimension Tables:
    - Dim_Date
    - Dim_Room
    - Dim_Customer
    - Dim_HotelBranch
- Created basic visualizations for validation
